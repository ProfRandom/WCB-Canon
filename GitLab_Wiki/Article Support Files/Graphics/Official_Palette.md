# WCB Canon — Accessible 9-Color WONG Palette

![WCB_t0_Color.png](./WCB_t0_Color.png)

## Color Definitions

| Name | Hex Code |
|:--|:--|
| WCB_Silver | `#ADADAD` |
| WCB_Grey | `#808080` |
| WCB_Orange | `#E69F00` |
| WCB_Cyan | `#56B4E9` |
| WCB_Green | `#009E73` |
| WCB_Yellow | `#F0E442` |
| WCB_Blue | `#0072B2` |
| WCB_Red | `#D55E00` |
| WCB_Violet | `#CC79A7` |

**Notes**
- Palette is optimized for **color-blind accessibility** (passes protanopia, deuteranopia, tritanopia checks).  
- Warm/cool hues follow an approximately **137° golden-ratio distribution** for visual balance.  
- Performs consistently across **CMYK print**, **sRGB screen**, and **LaTeX vector** outputs.
